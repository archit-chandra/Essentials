Spring Annotations
==========================================================
@Component         : to register this bean as component Superclass of @Controller, @Service
@Controller        : register class as bean and controller in Spring MVC
@RequestMapping    : to map methods to http request paths
    @GetMapping
    @PostMapping
    @PutMapping
    @DeleteMapping
@Autowired         : to inject the dependency
    @Qualifier     : When more than one implementation, call one of the implementation from (@Qualifier("bean_name")
@Service           : to know this bean is a service component
    @Primary       : When more than one implementation, make one implementation as default
@Entity
